## Language Systems

### Amtreadan (Elven)

Use for names with Elvish origins (Sindarin/Quenya). Characteristics:

- Agglutinative structure (compound words from root concepts)
- Uses diacritics: ë, ú, í, î, ô, á, é, ó, ä, ö
- Common suffixes: -ion, -iel, -wen, -dor, -lond, -ost, -rim, -roth
- Prefixes: A- (negation), U- (temporary), I-/O- (proximity/distance)
- See: `elven-vocab.md` and `elven-grammar.md`

### Ancient Human (Old English/Celtic)

Use for names with Mannish/Human origins. Characteristics:

- Old English/Norse/Celtic style
- Examples: Rohan → Aranada, Gondor → Argenthal
- Common patterns: -helm, -wyn, -gar, -hold, -land

### Dwarvish

Use for Dwarf-related names. Characteristics:

- Guttural consonants: kh, gh, zh
- Common elements: -dûm, -zâram, -zigil, -khâd
- Example: Khazad-dûm → Shadowdeep (translated)
